# PlaneChat App

PlaneChat is an iOS application that allows users to chat with others without an
internet connection, using the [Multipeer Connectivity
Framework](https://developer.apple.com/documentation/multipeerconnectivity).

---
## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo run:ios
   ```
(An iOS development environment is required to run the app on a simulator).
---
## Technologies Used
- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [Multipeer Connectivity
  Framework](https://developer.apple.com/documentation/multipeerconnectivity)
